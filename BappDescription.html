This extension provides an additional passive Scanner check for metadata in PDF files.

Requires Jython 2.7, pdfminer and chardet.

Some installations of Python don't include pdfminer or chardet. In this case, you need to download them from their official sites:

  • pdfminer: https://pypi.python.org/pypi/pdfminer/
  • chardet: https://pypi.python.org/pypi/chardet/
You will need to specify the location of the directory containing the packages in Extender / Options / Python Environment / Folder for loading modules.
